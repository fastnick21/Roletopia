DEVELOPMENT HISTORY

CHANGELOG

Roletopia separates completed foundation work from planned gameplay features, so visitors can see exactly what exists and what remains under development.

WEBSITE

July 2026

Complete website first release

Converted the original single-page mockup into a responsive multi-page project hub.

  • Finished the animated landing page and mobile navigation
  • Added role encyclopedia, download center, documentation, and changelog
  • Added truthful release status and live GitHub download totals
  • Added accessibility improvements and reduced-motion support
CORE

July 2026

Portable framework hardening

Strengthened the reusable code before beginning direct Among Us integration.

  • Moved portable libraries to .NET Standard 2.0
  • Added player, connection, alive-state, task, meeting, and vote validation
  • Added tie, skip, and ejection result handling
  • Made the outbound network queue thread-safe
PLUGINS

July 2026

Safer manifest discovery

Improved the plugin foundation while keeping executable loading outside the unfinished API.

  • Validates plugin IDs, names, versions, and role lists
  • Rejects duplicate plugin identities
  • Skips malformed, unreadable, or inaccessible manifests safely
CI

July 2026

Cross-platform build checks

Added automated checks to catch regressions before a release is created.

  • Builds portable components on Windows and Ubuntu
  • Runs engine and networking smoke tests
  • Builds the Windows installer separately
  • Checks release files before publishing tagged builds
