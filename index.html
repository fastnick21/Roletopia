FAQ

Common questions

What does the counter track?

It totals downloads of assets attached to published GitHub releases. It does not invent an active-player count.

Do all players need the mod?

Roletopia is being designed as a host-focused mod. Actual compatibility will depend on the completed game integration and each release.

Where can I report a problem?

Use the Support link to open an issue in the Roletopia GitHub repository.
