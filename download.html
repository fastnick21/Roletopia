INSTALLATION

How releases will work

These steps apply after a playable installer is attached to an official release.

STEP 1

Verify the release

Use only the official GitHub release linked from this page. Check the version notes and supported Among Us version.

STEP 2

Close Among Us

Exit the game completely, keep the installer next to its packaged Roletopia folder, and run it as your normal user.

STEP 3

Select the game folder

The installer checks common Steam locations or accepts the folder containing Among Us.exe.

Important: The current repository contains framework components, not a finished playable package. The website will not label it stable until integration and game testing are complete.

DOWNLOAD FAQ

Release safety

Why is the release button disabled?+

No completed playable release has been published yet. The button will link to the newest official release automatically when one exists.

Can I install the DLL artifacts directly?+

No. The portable framework DLLs are development outputs and do not include the required Among Us loader and runtime hooks.

Will Roletopia update automatically?+

Automatic updating is planned for a later launcher milestone. Early releases will use the official download page and version notes.
